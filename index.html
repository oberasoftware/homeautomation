What is Obera HomeAutomation?

Obera has built a user-friendly home automation solution that offers ease of use for all your home automation appliances. We offer integration for multiple home automation hardware products and are still expanding with more options.

Widgets

HomeAutomation has a responsive dashboard with widgets representing the status of your devices.

Rules Editor

Obera HomeAutomation comes with a full fledged rules editor that allows you to create and edit rules around your devices

History graphs

We provide graph widgets showing things like power, light or temperature history of your sensors and devices.

Mobile-friendly

Our interface is designed to fully support mobile browsers using Bootstrap.

Extensible

We offer extension support to introduce new plugins and widgets

Virtual Items & Groups

Group your devices into logical groups or create virtual items to assign status to.

Features

  • Graph chart support
  • Rules editor
  • Fully Async & Event driven
  • Dimmer, switch and value Widgets

Supported hardware

  • Philips Hue
  • ZWave
  • Youless Energy monitor
  • Nest (in progress)

Configure ZWave

Edit file: configuration/application.properties and change zwave.serial.port to correct value: 
#Points to the serial port for the ZWave adapter
#Example values:
#  COM1 for Windows
#  /dev/ttyS0 (or similar) for Linux
#  /dev/ttyUSB0 for Linux
#  /dev/tty.SLAB_USBtoUART for OSX
zwave.serial.port=/dev/ttyS0
